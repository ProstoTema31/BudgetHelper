# BudgetHelper (Анализатор актов сверки)

Desktop-приложение для автоматической сверки бухгалтерских взаиморасчетов. Программа парсит выгрузки актов сверки, сопоставляет операции и генерирует наглядный Excel-протокол расхождений. 

## 🚀 Ключевые возможности

* **Мультиформатность:** Поддержка парсинга `Excel` и `PDF` (включая умное разделение двусторонних актов).
* **Интеллектуальное сравнение:** * Сопоставление сумм по модулю (Дебет одной компании = Кредит другой).
  * Окно допусков (5 дней), если платеж прошел с задержкой.
* **Отличная производительность:** Алгоритм блочной сортировки (Bucket Sort/ToLookup) обрабатывает десятки тысяч строк за доли секунды.
* **Наглядный экспорт:** Выгрузка результатов проверки в готовый `.xlsx` файл с цветовой индикацией совпадений, разрывов в датах и отсутствующих документов.

## 🛠 Стек технологий

* **C# / .NET** (Windows Forms)
* **EPPlus** — генерация и чтение таблиц Excel без установленного MS Office.
* **UglyToad.PdfPig** — геометрический разбор и извлечение данных из PDF.
* **Newtonsoft.Json** — промежуточное хранилище JSON.

## 💻 Как использовать

1. Перейдите во вкладку [Releases](../../releases) и скачайте последнюю версию программы.
2. Запустите приложение.
3. Выберите первый файл (акт вашей компании) и второй файл (акт контрагента). 
   > *Примечание: Если контрагент прислал единый двусторонний PDF-акт, просто выберите его в оба поля.*
4. Нажмите **«Сверить»**. Процесс отобразится на индикаторе загрузки.
5. Нажмите **«Экспорт в Excel»**, чтобы сохранить готовый протокол расхождений.
